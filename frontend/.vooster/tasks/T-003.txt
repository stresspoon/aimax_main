**Importance:** MUST

## Description

### 설명
대시보드에서 AI 글쓰기 화면으로 이동할 수 있는 진입점 버튼/카드를 구현하고, 나머지 기능 클릭 시 준비 중 모달을 띄웁니다.
### 구현 세부 사항
1. AI 글쓰기 카드에 Link 또는 useRouter를 사용한 이동 로직 추가
2. 나머지 카드 클릭 시 Tailwind 모달 컴포넌트로 "준비 중" 안내 표시
3. 모달 닫기 버튼 및 배경 클릭 시 모달 종료 처리
### 테스트 전략
- AI 글쓰기 카드 클릭 시 /ai-writing 경로로 이동 확인
- 다른 카드 클릭 시 준비 중 모달이 정상 노출되는지 검증
- 모달 닫기 동작 확인

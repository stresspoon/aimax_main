## Description

## 설명
SNS 검증 결과를 기반으로 신청자를 선정 또는 비선정 처리하고, 결과를 DB 및 구글시트에 기록합니다.
## 구현 세부사항
1. 선정 로직(하나라도 기준 충족 시 선정) 구현
2. 결과 저장을 위한 DB 테이블 설계 및 쿼리 작성
3. Google Sheets API로 결과 기록 업데이트 기능 구현
4. 백그라운드 작업 큐를 통한 비동기 처리
5. 에러 발생 시 롤백 및 알림 처리
## 테스트 전략
- 다양한 SNS 결과 케이스에 따른 선정/비선정 결과 확인
- 구글시트 결과 반영 정상 동작 검증
- 처리 중 예외 발생 시 트랜잭션 롤백 확인

# 메일 자동 발송 시스템 및 템플릿 관리 기능 구현

**Task ID:** T-013
**Status:** BACKLOG
**Importance:** MUST
**Complexity:** 7/10
**Urgency:** 9/10
**Dependencies:** T-012

## Description

## 설명
프로젝트 생성 4일 후 선정자/비선정자에게 자동으로 메일을 발송하고, 관리자가 템플릿을 직접 편집할 수 있는 기능을 구현합니다.
## 구현 세부사항
1. 메일 스케줄러 로직(Node Cron 또는 Bull 스케줄) 작성
2. SendGrid/AWS SES 연동 및 템플릿 저장용 DB 테이블 생성
3. 관리자 UI에 메일 템플릿 입력 폼 추가
4. 4일 후 발송 로직과 템플릿 병합 엔진 구현
5. 발송 내역을 DB에 기록하고 대시보드 API 제공
## 테스트 전략
- 스케줄러에 설정된 시점에 메일 정상 발송 확인
- 템플릿 변수 치환 및 포맷 검증
- 발송 실패 시 재시도 및 알림 동작 검증
